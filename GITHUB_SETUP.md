## Step 2: Push Your Code

After creating the repo, run these commands:

```bash
cd ~/projects/greenhouse-job-scraper
git remote add origin https://github.com/d-alleyne/greenhouse-job-scraper.git
git branch -M main
git push -u origin main
```

## Step 3: Verify

Visit https://github.com/d-alleyne/greenhouse-job-scraper to see your published code.
